#Setup

## Set permissions

chmod +x scripts/ralph/ralph.sh

## Start proxy

Den Proxy starten: Bevor du das Script ausführst, öffne ein zweites Terminal-Tab und starte den Proxy. Auf dem Mac kannst du seit dem Update von Januar 2026 auch das Dashboard nutzen:

Befehl: antigravity-claude-proxy start

Dashboard (im Browser): http://localhost:8080/dashboard (hier siehst du live deine Token-Auslastung bei Google).


### Let Ralph do your Job

./scripts/ralph/ralph.sh 10
